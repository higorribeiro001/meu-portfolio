Destaques:

API para previsão de ataque cardíaco

API Django Rest Framework com códigos curtos, precisos e eficiente, com uma rede neural desenvolvida com Deep Learning (Aprendizado Profundo). Esta API trabalha junto à rede neural buscando prever se o paciênte que forneceu os dados tem altas ou baixas chances de sofrer um ataque cardíaco.

Para acessá-lo no github: https://github.com/higorribeiro001/api-predicao-ataque-cardiaco.

API para previsão de previsão de desempenho de um estudante

API desenvolvida em Django Rest framework com rede neural com aprendizado profundo capaz de previsão de desempenho de um estudante utilizando regressão linear.

Para acessá-lo no github: https://github.com/higorribeiro001/api-performance-student.

Aplicativo para previsão de desempenho escolar de estudantes

Aplicativo mobile desenvolvido em React Native utilizando expo além de outros recursos do framework. Typescript também foi utilizado para tipagem de atributos, além do uso do framework Tailwind.css para estilizar os templates que evitam a criação de estilos e dão mais fluidez às telas.

Para acessá-lo no github: https://github.com/higorribeiro001/app_performance_student.
